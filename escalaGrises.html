Promedio RGB y luma

Escala de grises

Informe

Un pixel es la base fundamental (menor unidad) que componen una imágen. Cada uno de estos está representado por un modelo de color llamado RGB (Red, Gree, Blue - siglas en inglés), mediante el cual es posible representar un color mediante la mezcla de los tres colores de luz primarios (RGB).

Cuando se tiene una imagen a color, es posible convertirla a escala de grises, es decir, todos sus colores son remplazados por el color negro (en diferentes intencidades) y blanco. Para esto, se utilizan dos metodos escenciales; promedio RGB y LUMA (o método de luminosidad).

Promedio RGB

Este método consiste en extraer los valores individuales de cada color del pixel seleccionado. Luego, debido a que estos valores son numéricos (cada uno variando de 0 a 255), es posible realizar un promedio entre estos valores, así;

promRGB = (rojo + verde + azul)/3

Por último, es valor numérico del color del pixel es actualizado al nuevo promRGB. El problema acà radica en que cada color primario aporta en igual proporción al color final del pixel. A esta propoción se le llama "luminosidad", y como veremos mas adelante, cada color primario aporta un nivel de luminosidad diferente, cada uno tiene su propia contribución en la construcción de una imágen.

todos estos valores siempre será igual a 1.

LUMA o método de luminosidad

Este método resuelve el problema que se tiene con el Promedio RGB, pues cada color primario tiene su propio aporte en la imagen, con diferentes niveles de luminosidad. De esta forma, el nuevo color de cada pixel esta dado en estas proporciones:

LUMA = 0.299 * rojo + 0.587 * verde + 0.114 * azul

La proporción de luminosidad varía dependiendo al estandar manejado, que a su vez varía dependiendo del proposito con el cual se construye. (Se escoge el presente para propositos practicos)
